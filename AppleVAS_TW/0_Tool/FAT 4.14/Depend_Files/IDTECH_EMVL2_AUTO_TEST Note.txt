1. Script Format:
	EMV_L2_START: < Message >        //Case Start + <info for the below test>
	CMD:[ Hex Command without protocal paked ] // Configuration Command
	...
	RCM:[ Hex Command with protocal paked ] // Configuration Command with raw format
	...
	
	MESSAGE:[ Message ! ] // Info befor EMV L2 Trans Process
	
	START TRANS:[ Start Trans Hex Command, without protocal paked ] //Start Trans 
	AUTHENTICATE:[ Authenticate Hex Command, without protocal paked ] //Authenticate
	ENTER PIN:[ Enter PIN Block Hex Command, without protocal paked ] //Enter PIN
	APPLY HOST:[Complete Transaction Hex Command, without protocal paked ] //Complete Transaction
	MSR FALLBACK:[ Msr Block Hex Data ] // MSR Fallback
	RETRIVE DATA:[ Retrive Data Hex Data ] // Retrive data
	
	EXPECT_TRANS:[ Some Expect for Start Trans ]
	EXPECT_AUTHENTICATE: [ Some Expect for Authenticate ]
	EXPECT_ENTERPIN:[ Some Expect for Enter PIN ]
	EXPECT_APPLYHOST:[ Some Expect for Complete Transaction ]
	EXPECT_RETRIVEDATA:[ Some Expect for Retrive data]
	
	EMV_L2_END:                       //Case end
